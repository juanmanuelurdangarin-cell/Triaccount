⚙️ Configurar Firebase

Para usar cuentas de Google necesitás conectar tu propio proyecto Firebase gratuito. Solo se hace una vez.

Pasos rápidos:
1. Ir a console.firebase.google.com
2. Crear proyecto → Agregar app Web
3. Copiar los datos del SDK aquí abajo
4. En Authentication → Habilitar Google
5. En Firestore → Crear base de datos

¿No querés configurar Firebase? Usar modo local (sin cuentas)
